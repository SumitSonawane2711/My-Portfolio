# Basics of React

React is a popular JavaScript library for building user interfaces, especially single-page applications. It allows developers to create reusable UI components and efficiently update the DOM.

## Key Concepts

- **Components:** The building blocks of React applications. Components can be functional or class-based.
- **JSX:** A syntax extension that lets you write HTML-like code in JavaScript.
- **Props:** Short for "properties", props are used to pass data from parent to child components.
- **State:** State is used to manage data that changes over time within a component.
- **Lifecycle Methods:** Special methods in class components that allow you to run code at specific points in a component's life.

## Example

```jsx
import React, { useState } from 'react';

function Counter() {
    const [count, setCount] = useState(0);

    return (
        <div>
            <p>You clicked {count} times</p>
            <button onClick={() => setCount(count + 1)}>
                Click me
            </button>
        </div>
    );
}
```

## Getting Started

1. Install React using `npm`:
     ```
     npm install react react-dom
     ```
2. Create components and use them in your application.
3. Use tools like Create React App for quick setup.

React makes it easy to build interactive and dynamic web applications.
