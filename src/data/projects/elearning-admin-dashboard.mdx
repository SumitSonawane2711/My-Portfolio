## Project Overview

The eLearning Platform is a web-based learning management system (LMS) that enables learners to access online courses, track their learning progress, and interact with educational content through an intuitive interface.

The platform provides administrators with tools to manage courses, users, and learning materials, while offering students a seamless experience for exploring and completing courses. It was built as a scalable full-stack application with a focus on performance, usability, and maintainability.
