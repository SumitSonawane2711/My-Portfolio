## Project Overview

Rwaltz Web Platform is a corporate website and content management system built for a software development and Web3 consultancy. The platform showcases the company's services, portfolio, case studies, blogs, careers, and client testimonials while providing an administrative dashboard for managing all website content.

The project consists of a dynamic marketing website, a powerful admin panel, and a centralized backend API, enabling non-technical users to manage website content without requiring developer intervention.

---

## My Role

- Developed the backend REST APIs from scratch using AdonisJS and TypeScript.
- Designed and implemented the database schema for the CMS and business modules.
- Built the complete admin panel for content management.
- Implemented role-based authentication and fine-grained permission management.
- Developed CRUD APIs for projects, blogs, case studies, testimonials, careers, FAQs, newsletters, and enquiries.
- Integrated AWS S3 for media uploads and file management.
- Implemented transactional email functionality.
- Converted the static marketing website into a CMS-driven platform through API integration.
- Built reusable service modules for frontend API communication.
- Collaborated on frontend development and ensured seamless integration between backend and both React applications.
