## Project Overview

SnapGram is a full-stack social media application that allows users to create and share posts, connect with other users, and engage through likes, saved posts, and real-time interactions. The platform focuses on delivering a modern, responsive, and engaging user experience with secure authentication and smooth content management.

---

## Highlights

- Built a complete social media platform from the ground up.
- Implemented secure user authentication with sign-up and sign-in functionality.
- Developed features for creating, editing, deleting, and managing posts.
- Built user profile management with customizable profile information.
- Added post interactions including likes and saved posts.
- Implemented chat functionality for user communication.
- Integrated infinite scrolling for a seamless content browsing experience.
- Designed responsive and reusable UI components for a consistent user experience.
