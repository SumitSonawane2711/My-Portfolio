# First Project Video

This project focuses on presenting video content in a clean and accessible way. The goal was to keep the browsing experience simple while making the featured media feel polished across screen sizes.

## Highlights

- Built a responsive detail page for project storytelling.
- Kept the card description short while moving deeper notes into MDX.
- Used reusable portfolio sections so the project can grow without changing the page structure.

## Technical Notes

The project detail is powered by an MDX file, which makes it easier to add long-form content, lists, code snippets, and future media without bloating the project constants.
